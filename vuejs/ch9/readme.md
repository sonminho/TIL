# _단일 파일 컴포넌트_


### 전역 수준 컴포넌트의 문제점
- 빌드 단계가 없으므로 ES6, TypeScript 같은 자바스크립트 문법을 사용할 수 없음
- CSS 스타일을 빌드하고 모듈화할 수 있는 기능을 제공하지 않음
- HTML 파일안에 여러 개의 ```<template/>``` 태그가 작성되어 식별하기가 어려움


### Vue CLI 로 프로젝트 생성하기
> vue create [project name]  
생성된 프로젝트는 3가지 영역으로 나뉘어 있음
1. ```<template/>```
2. ```<script/>```
3. ```<style/>```
